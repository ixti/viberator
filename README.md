# Viberator

Viberator (prononce as in `vibe` and then `rator`) is a small utility to
generate HTML dums of your ViberPC chat logs.

## Installation

Add this line to your application's Gemfile:

    gem 'viberator'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install viberator

## Usage

    $ viberator
